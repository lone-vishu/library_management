# Library Management System (Python Project)

A command-line based Library Management System built using Python to manage book records and basic library operations.

This project allows users to:
- Add books
- View available books
- Search books
- Borrow books
- Return books

The project is designed to strengthen Python fundamentals and simulate real-world library management operations.

---

## 📌 Project Overview

The Library Management System provides a menu-driven interface for handling book-related operations.

### Features:
- Add new books
- Store:
  - Book Title
  - Author Name
  - Number of Copies
- View all available books
- Search books by:
  - Book Name
  - Author Name
- Borrow books
- Return books
- Update available copies automatically
- Handle invalid inputs using exception handling
